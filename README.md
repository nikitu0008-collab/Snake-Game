> Классическая игра «Змейка», реализованная на Python с использованием библиотеки Pygame.  
> Управляйте змейкой, собирайте еду и становитесь чемпионом по длине!
---

## 🎮 Игровой процесс

- **Цель** — съесть как можно больше зелёных яблок.
- С каждым съеденным яблоком змейка **растёт** на один сегмент.
- Игра заканчивается, если змейка **врезается в стену** или **в свой собственный хвост**.
- Текущий счёт отображается в верхней части экрана.

### Управление
| Клавиша | Действие |
|---------|----------|
| `↑ w`| Движение вверх |
| `↓ s`| Движение вниз |
| `← a`| Движение влево |
| `→ d`| Движение вправо |
| `C`| Продолжить |
| `Q`| Выход

## 🚀 Установка и запуск

1. **Клонируйте репозиторий**
   git clone https://github.com/ваш_ник/snake-game.git
   cd snake-game
2. Установите зависимости (требуется Pygame)
    
    pip install pygame

    Запустите игру

    python src/main.py
