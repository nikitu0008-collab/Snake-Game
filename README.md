## 🎮 Игровой процесс

- **Цель** — съесть как можно больше красных яблок.
- С каждым съеденным яблоком змейка **растёт** на один сегмент.
- Игра заканчивается, если змейка **врезается в стену** или **в свой собственный хвост**.
- Текущий счёт отображается в верхней части экрана.

### Управление
| Клавиша | Действие |
|---------|----------|
| `↑` / `W` | Движение вверх |
| `↓` / `S` | Движение вниз |
| `←` / `A` | Движение влево |
| `→` / `D` | Движение вправо |
| `Пробел` | Пауза / Продолжить |

## 🚀 Установка и запуск

1. **Клонируйте репозиторий**
   git clone https://github.com/ваш_ник/snake-game.git
   cd snake-game
2. Установите зависимости (требуется Pygame)
    
    pip install pygame

    Запустите игру

    python src/main.py
